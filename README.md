# The Phone Cave
The Phone Cave is a web application that offers the best smartphones for the best prices

## Setup
* Server:

```
$ npm install
$ npm run dev
(It is executed on port 5000)
```

* Client:

```
$ npm install
$ npm start
(It is executed on port 3000)
```
